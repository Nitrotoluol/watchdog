Устройство для автоматической перезагрузки компьютера в случае зависания и управление устройствами, благодаря управляемому ЮСБ и гальванически развязанным управлением питания

Программное обеспечение адаптировано для работы на контроллере STM32f301С8, однако с легкостью может быть переделано пол любой программируемый контроллер.

запуск в CooCox IDE:
1. Открыть проект \SOFTWARE\CooCox\HW_WD.coproj
2. Перекомпилировать проект Project -> Rebuild  (CTRL+R) 
3. Запись прошивки в контроллер Flash -> Program Download
